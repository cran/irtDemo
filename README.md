# Item Response Theory Demo Collection

The package can be installed and loaded as:
```{r}
install.packages("irtDemo")
library(irtDemo)
```
Example usage:
```{r}
irtDemo("eapmap")
irtDemo("mirt")
irtDemo("mle")
irtDemo("est2pl")
irtDemo("est3pl")
```
Shiny web applications:
